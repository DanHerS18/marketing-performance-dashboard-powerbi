# Marketing Performance Dashboard (Power BI)

This project showcases a multi-channel marketing performance dashboard built in Power BI.

## Overview
The dashboard provides a high-level view of marketing performance across channels and data sources, focusing on spend efficiency, engagement, and conversions.

## Key Features
- Top KPI cards (Spend, CPM, CTR, CPC) with trend indicators
- Channel Performance analysis
- Data Source and Campaign Performance tables
- Time series visualization by marketing channel

## Data
The dataset was generated using synthetic monthly marketing data to simulate realistic marketing behavior across multiple channels and platforms.

## Files
- `Marketing_Performance_Dashboard.pbix`: Power BI Desktop file
- `Marketing_Performance_Dashboard.pdf`: Exported dashboard for easy viewing
- `marketing_performance_dummy_data.csv`: Dummy dataset used for the dashboard

## Tools Used
- Power BI Desktop
